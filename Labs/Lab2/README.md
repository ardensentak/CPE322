## Command Line 
### Instructions:
Go to the IOT repository. Review Lessons 1 and 2. Then, open a terminal and enter the following commands:</br>
`hostname`, 
`env`, 
`ps`, 
`pwd`, 
`git clone https://github.com/kevinwlu/iot.git`, 
`cd iot`, 
`ls`, 
`cd`, 
`df`, 
`mkdir demo`, 
`cd demo `, 
`nano file`, 
`cat file`, 
`cp file file1`, 
`mv file file2`, 
`rm file2`, 
`clear`, 
`man uname`, 
`uname -a`, 
`ifconfig`, 
`ping localhost`, 
`netstat`. 
Document results to GitHub repository.
